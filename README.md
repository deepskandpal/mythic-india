# Mythic India

Browser-based open world adventure game exploring Indian mythology.

## Concept

Episodic gameplay — new world/story arc unlocked biweekly or monthly. Each episode delivers 4-6 hours of gameplay built around a mythology arc. AAA storytelling ambition, indie execution.

Players explore, discover, and live the stories of Indian mythology through gameplay — not cutscenes.

## Episodic Arcs (planned)

- Samudra Manthan (Churning of the Ocean)
- Vikram-Betaal (25 riddle stories)
- Rama's Exile (Aranya Kanda)
- Dashavatara (10 avatars, 10 episodes)
- Ashoka's Conversion
- Panchatantra fables
- ...and more
